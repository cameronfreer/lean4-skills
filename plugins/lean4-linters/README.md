# lean4-linters

Lean 4 linter patterns for project-specific rules, with templates for options, syntax traversal, and clean error spans.

## What you get

- Linter templates with `register_option` and `addLinter`
- Guidance for warnings vs errors
- Project-wide enablement and local disable patterns
