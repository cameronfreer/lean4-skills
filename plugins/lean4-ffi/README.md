# lean4-ffi

Lean 4 FFI patterns for binding C/ObjC libraries, with struct layout and Lake linking guidance.

## What you get

- Minimal `@[extern]` binding patterns
- `@[cstruct]` layout guidance
- Lake `extern_lib` linking template
