# lean4-metaprogramming

Lean 4 metaprogramming patterns for building DSLs, macros, elaborators, and custom pretty-printing.

## What you get

- Decision tree for macro vs elaborator vs pretty-printer
- Templates for syntax categories, validation, and hygiene
- Debugging checklist (macro expansion, elaboration, printing)
